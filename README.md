# 🧠 درباره‌ی یادگیری عمیق (Deep Learning)

**یادگیری عمیق** شاخه‌ای از یادگیری ماشین است که با استفاده از شبکه‌های عصبی چندلایه، الگوهای پیچیده را از داده‌های خام استخراج می‌کند. این حوزه انقلابی در زمینه‌های مختلف از جمله **بینایی کامپیوتر، پردازش زبان طبیعی، تشخیص صدا، خودروهای خودران و پزشکی** ایجاد کرده است.

---

## 🎯 چرا یادگیری عمیق؟

- **یادگیری خودکار ویژگی‌ها**: برخلاف روش‌های سنتی، نیازی به استخراج دستی ویژگی‌ها نیست.
- **عملکرد بالا با داده‌ی زیاد**: هرچه داده بیشتر باشد، مدل‌های عمیق بهتر عمل می‌کنند.
- **قابلیت تعمیم‌پذیری**: مدل‌های عمیق می‌توانند روی داده‌های جدید نیز عملکرد خوبی داشته باشند.

---

## 📁 این ریپازیتوری چیست؟

این مخزن شامل مجموعه‌ای از پروژه‌های عملی برای یادگیری و پیاده‌سازی مفاهیم یادگیری عمیق با استفاده از **TensorFlow** و **Keras** است.

### پروژه‌های فعلی:

1. **طبقه‌بندی تصاویر Fashion MNIST**  
   - شناسایی ۱۰ نوع لباس با استفاده از شبکه‌های عصبی عمیق  
   - استفاده از تکنیک‌های Dropout و EarlyStopping  

2. **پیش‌بینی عددی (رگرسیون)** (به زودی)  
   - پیش‌بینی مقادیر پیوسته مانند قیمت مسکن یا دما  

---

## 🛠️ ابزارهای استفاده‌شده

- **TensorFlow 2.x**: کتابخونه‌ی اصلی برای ساخت و آموزش مدل‌ها
- **Keras**: رابط کاربری سطح بالا برای TensorFlow
- **Matplotlib**: برای رسم نمودارهای آموزشی
- **Jupyter Notebook / Google Colab**: محیط اجرای کدها

## 🚀 مسیر یادگیری

اگر تازه‌کار هستید، پیشنهاد می‌کنم به ترتیب زیر پیش بروید:

1. آشنایی با پایتون و کتابخانه‌های NumPy و Pandas
2. یادگیری مبانی یادگیری ماشین (رگرسیون، طبقه‌بندی، ارزیابی مدل)
3. شروع با شبکه‌های عصبی ساده (MLP)
4. ورود به شبکه‌های کانولوشنی (CNN) برای پردازش تصویر
5. آشنایی با شبکه‌های بازگشتی (RNN) برای داده‌های دنباله‌ای
